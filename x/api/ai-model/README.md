# AI Model Runtime

This folder is used by the Tiwlo X backend AI module.

- Engine: `node-llama-cpp`
- Default model: `Qwen/Qwen1.5-1.8B-Chat-GGUF`
- Default file: `models/qwen-1.5-1.8b/qwen1_5-1_8b-chat-q4_k_m.gguf`

Large `.gguf` model files are intentionally ignored by git. Use the administrator AI Model page or the backend download mutation to fetch the model on each server that should run local AI.
